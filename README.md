# 🎮 Tic Tac Toe - Premium Web Edition

A modern, responsive, and visually stunning web-based implementation of the classic Tic Tac Toe game. This project focuses on clean code architecture, robust game state management, and a premium "Cyberpunk-inspired" user interface.

## ✨ Features
* **Interactive UI:** A sleek, dark-themed interface with smooth hover transitions and neon glow effects.
* **Game Logic:** Efficient state management to handle player turns, win condition validation (rows, columns, and diagonals), and draw scenarios.
* **Responsive Design:** Built with CSS Grid to ensure an optimal experience across both desktop and mobile devices.
* **Visual Feedback:** Color-coded markers for Player X (Neon Pink) and Player O (Neon Cyan) to enhance gameplay clarity.
* **Performance:** Lightweight, zero-dependency project using pure HTML, CSS, and Vanilla JavaScript.

## 🛠️ Tech Stack
* **Markup:** HTML5 (Semantic Structure)
* **Styling:** CSS3 (Flexbox & CSS Grid)
* **Scripting:** Vanilla JavaScript (ES6+)
